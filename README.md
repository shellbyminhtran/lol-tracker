
================================================================================
                    HUONG DAN SU DUNG LOL COOLDOWN TRACKER
================================================================================

Chuc mung ban da so huu phan mem ho tro tinh toan thoi gian hoi chieu (Cooldown)
danh rieng cho Lien Minh Huyen Thoai. Day la cong cu thuc chien giup ban kiem
soat tran dau tot hon.

1. YEU CAU HE THONG (Doi voi nha phat trien)
--------------------------------------------------------------------------------
De chay hoac dong goi lai ung dung tu ma nguon, may tinh can cai san:
- Node.js (Phien ban moi nhat)
- Git (De quan ly va tai ma nguon tu GitHub)

2. CAI DAT VA KHOI CHAY LAN DAU
--------------------------------------------------------------------------------
Neu ban tai code tu GitHub ve mot may tinh moi:
- Buoc 1: Mo Terminal tai thu muc chua code.
- Buoc 2: Go lenh `npm install` de tu dong tai lai cac thu vien can thiet.
- Buoc 3: Go lenh `npm start` de khoi chay ung dung o che do xem truoc.

3. HUONG DAN SU DUNG TRONG TRAN DAU
--------------------------------------------------------------------------------
- Chon phep: O moi vi tri (Top, Rung, Mid, ADC, SP), chon 2 phep bo tro tuong
  ung ma doi thu dang cam.
- Tuy chinh Haste: Chon muc "Haste" neu doi thu co Giay Khai Sang Ionia (+12)
  hoac ngoc Thau Thi Vu Tru (+18). App se tu dong tinh toan thoi gian thuc te.
- Kich hoat: Click truc tiep vao hinh anh Icon cua phep do khi doi thu su dung.
- Theo doi: So giay con lai se hien thi ngay tren Icon. Khi hien chu "XONG",
  phep do da san sang.

4. CACH DONG GOI THANH FILE .EXE
--------------------------------------------------------------------------------
De tao ra mot file .exe doc lap khong can mo code:
- Buoc 1: Mo Terminal tai thu muc APP.
- Buoc 2: Go lenh `npm run dist`.
- Buoc 3: Mo thu muc `dist/` vua xuat hien, ban se thay file .exe cua minh.
- Buoc 4: Chuot phai vao file .exe va chon "Pin to Taskbar" de su dung nhanh.

5. LUU Y QUAN TRONG
--------------------------------------------------------------------------------
- Ket noi mang: Ung dung can ket noi Internet luc moi khoi dong de tai du lieu
  moi nhat tu Riot Data Dragon.
- Riot Vanguard: Day la cong cu ho tro thui cong (Manual), hoan toan khong can
  thiep vao bo nho game, vi vay an toan tuyet doi voi he thong chong gian lan.
- Bao mat: Tuyet doi khong chia se file code co chua API Key ca nhan len cac
  dien dan cong khai.
